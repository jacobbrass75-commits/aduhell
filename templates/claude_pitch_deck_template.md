# Pitch Deck Template

Use this structure for a homeowner-facing ADU pitch deck:

1. Title slide
2. Property overview
3. Lot and zoning context
4. Likely ADU paths
5. Recommended concept path
6. Concept visual slide
7. Constraints and diligence slide
8. Next-step CTA slide

Rules:
- Use only approved facts from the property brief.
- Treat all renders as conceptual / illustrative only.
- Surface uncertainty where parcel geometry, overlays, transit, or garage evidence are not confirmed.
